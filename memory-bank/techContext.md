# Technical Context

## Tech Stack
- **Frontend**: Next.js, TypeScript, React
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Deployment**: Vercel (frontend), Docker (backend)

## Architectural Decisions
- Monorepo structure for better code sharing and collaboration.
- Modular design to ensure scalability and maintainability.
- Use of TypeScript for type safety and improved developer experience.

## Technical Constraints
- Ensure compatibility with existing tools and APIs.
- Optimize for performance and scalability.
- Maintain a consistent coding style and adhere to best practices.
