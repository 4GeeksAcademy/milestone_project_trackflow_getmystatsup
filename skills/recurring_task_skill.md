# Recurring Task Skill

## Objective
Automate the process of updating the `progress.md` file with the latest development updates.

## Inputs
- A summary of changes made in the current development cycle.
- The current date.

## Acceptance Criteria
- The `progress.md` file is updated with a new entry under the `Next Steps` section.
- The entry includes the date and a clear description of the changes.
- No existing content in the file is overwritten or lost.
