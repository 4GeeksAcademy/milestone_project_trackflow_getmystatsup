# TrackFlow Management System

A TypeScript-based management system for TrackFlow, covering:
- Warehouse Operations
- Carrier Management
- Reverse Logistics
- Customer Experience
- Client Relations
- Executive Dashboards

## Stack
- **Frontend:** React + TypeScript (Vite)
- **Backend:** Node.js + Express + TypeScript

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

---

This monorepo structure allows for scalable development of TrackFlow's management platform.
