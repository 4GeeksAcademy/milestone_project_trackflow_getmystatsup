# TrackFlow Website (TypeScript + React + Vite)

A professional, accessible, and SEO-optimized public website for TrackFlow, built with TypeScript, React, Vite, and Tailwind CSS.

## Features
- Responsive, accessible, and fast
- Semantic HTML and ARIA
- SEO meta tags and Schema.org JSON-LD
- Lead capture form with strict validation
- Multilingual-ready (base language: English)

## Getting Started

1. `npm install`
2. `npm run dev`

---

This project implements all requirements from typescript.md, webfundamentals.md, and seo_geo.md.
